ℹ️ A note about Anthropic in the browser

Anthropic's API doesn't currently allow direct requests from browser-based apps — this is a security feature called CORS (Cross-Origin Resource Sharing). When you use the Anthropic provider here, your request needs to go through a small server-side helper to reach the API.

  • Run a local proxy — Start a lightweight local proxy server (instructions below) that forwards requests from this page to Anthropic.
  • Use OpenAI instead — OpenAI's API supports direct browser requests and works without a proxy.
